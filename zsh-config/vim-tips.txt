diw — 删当前单词
ci" — 改引号内内容
ci( — 改括号内内容
ca{ — 改花括号及内容
dt{char} — 删到某字符之前
ct{char} — 改到某字符之前
Ctrl+d / Ctrl+u — 半屏下/上翻（替代疯狂按 j）
w / b — 按词前进/后退
e — 跳到当前词尾
f{char} — 行内跳到字符上，; 重复
s{c}{c} — flash.nvim 全屏跳转（两字符）
o / O — 下方/上方新建行并插入
A — 跳到行尾并插入
I — 跳到行首并插入
yiw — 复制当前单词
yi" — 复制引号内内容
p / P — 粘贴到光标后/前
dd — 删除整行
cc — 删除整行并插入
C — 删除到行尾并插入
D — 删除到行尾
J — 合并下一行
u / Ctrl+r — 撤销 / 重做
. — 重复上一次操作（超强）
>> / << — 缩进 / 反缩进
% — 跳到匹配括号
* / # — 搜索光标下单词（下/上）
gg / G — 文件头 / 文件尾
{数字}G — 跳到指定行
{ / } — 按段落跳
gg=G — 全文重新缩进
gd — 跳到定义（LSP）
gr — 查看引用（LSP）
K — 悬浮文档（LSP）
ys{motion}" — surround 加引号，如 ysiw"
ds" — 删除周围引号
cs"' — 双引号换单引号
[b / ]b — 上/下一个 buffer
za — 切换折叠
zR / zM — 展开/折叠全部
